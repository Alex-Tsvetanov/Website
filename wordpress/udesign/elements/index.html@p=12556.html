Glossary

Post Element Options

Customize Elements Globally or Individually

  • Select Posts: Choose posts by name.
  • Select Categories: Choose posts by category.
  • Posts Count: Control number of showing posts.
  • Order By: Choose order item.
  • Order Way: Choose order way.
  • Posts Layout: Choose layout for posts.
  • Image Size: Choose a size of image.
  • Vertical Align: Control vertical alignment of products layout.
  • Prevent Box Shadow Clip: Toggle to prevent that shadow effect is clipped.
  • Row: Choose row count from 1 to 6.
  • Columns: Choose column count on responsive mode.
  • Follow Theme Option: Toggle to decide whether you use product type which is set in theme option or not.
  • Content Options: Styling options for post content.
  • Button Style: Styling options for button.
