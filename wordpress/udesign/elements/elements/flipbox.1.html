Glossary

Flipbox Element Options

Customize Elements Globally or Individually

  • Choose Icon – Choose Icon from icon library or upload svg you made.
  • Title – Input flipbox title.
  • Subtitle – Input flipbox subtitle.
  • Content – Input flipbox description.
  • Button text – Input flipbox button label.
  • Button Link – Input flipbox button url.
  • Alignment – Control the horizontal alignment of flipbox content.
  • Max Height – Control the max height of flipbox.
  • Min Height – Control the min height of flipbox.
  • Animation Effects – Choose one flip effect you want to vary your approach to design.
  • Content Alignment – Control the vertical alignment of flipbox content.
  • Layout Background Type – You can set background type of flipbox as 2 types – classic and gradient.
  • Icon View – Select flipbox icon type from 3 types.
  • Icon Color – Control flipbox icon color.
  • Icon Size – Control flipbox icon size.
  • Icon Padding – Control flipbox icon padding.
  • Button Type – Select your favourite button type from 5 types.
  • Button Size – It can be small, medium, large and extra large.
  • Show icon – Enable to show or hide icon beside button label.
  • Icon Position – Control the icon position inside button.
  • Icon Spacing – Control the spacing between button label and icon.
  • Icon Hover Effect – Control the icon effect on button hover event.
