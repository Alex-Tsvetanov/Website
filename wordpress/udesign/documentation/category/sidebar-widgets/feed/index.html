UDesign offers a very easy way to add left, right or dual sidebars to any pages of your web site. You can show your content by adding widgets to widget area. If you want to use another sidebar for the other pages, or custom post type pages, it’s also possible by using sidebars builder and layout builder.

Sidebars provided by UDesign

  • Shop sidebar for all shop pages
  • Blog sidebar for all blog pages
  • If you import demo content, product sidebar will be added for product detail pages.
  • You can add more sidebars if you want.

Two sidebar types – Classic, Off-canvas

However, if you want more sidebars, the theme allows you to register new sidebar.

  1. Please go to sidebars builder page in UDesign > Sidebars menu.
  2. Please click “Add New Sidebar” button in top right area to register a new sidebar.
  3. Please enter your new sidebar’s name on input field.
  4. Please go to Apperance > Widgets page, and drag some widgets to your new sidebar as you need.
  5. You can display the sidebar by using layout builder. Read more.

Learn more details from our guided video.

To display sidebar in home page, please see the following guide.

  1. Please go to UDesign > Layout Builder in admin menu.
  2. Please select a layout which you want to display sidebar. You can also create a layout and set display condition. Please see about Layout Builder.
  3. please select the sidebar you have been made, and set your sidebar width in pixels or rem, or %, etc. The sidebar can be displayed in classic or off-canvas mode.
  4. By setting options of layout’s display condition, you can display the sidebar anywhere you want.

Please select “All shop pages” if you want to display the sidebar in shop page, or select specific product categories if you want.

All widgets are found in the Appearance > Widgets tab on your WP admin sidebar. There are various customization options and fields for each individual widget, and they are all self explanatory. Here are widgets that our Theme offers.

  1. UDesign – Block : Just use your blocks which are made by our Block Builder.
  2. UDesign – Posts : Display site posts ordered by ID, date and etc.
  3. UDesign – Price Filter : Please insert price filter to sidebar.
  4. UDesign – Products : Show off the products ordered by ID, date and etc.
  5. UDesign – Product Brands: Products brands are appeared into sidebar container.
  6. UDesign – Filter Clean : Clean all filters already chosen.
