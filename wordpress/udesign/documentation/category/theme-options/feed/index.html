There are 2 major groups of options: Theme Option and Layout Builder Option.

Here you can get detailed description about Theme Options.

Please navigate to UDesign > Theme Options on WP Admin sidebar or WP Admin toolbar.

UDesign Theme Option is the heart of UDesign options. They are presented in a well-organized panel, covering all areas of the theme. There are over 300 individual options here. These are global options that affect your entire site. You can set global site options easily with preview changes of site according to your setting.
