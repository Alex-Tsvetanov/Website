Instead, you can create your pages using our Elementor Page Builder elements (shortcodes) allowing to create page templates of any design. UDesign has developed an integrated set of elements expanding and improving options in order to provide comprehensive tools to present WooCommerce products and functionality.  

UDesign Header Elements

  1. Account – Allows you to place your account element on any part of your header.
  2. Cart – Offers you 4 kinds of display modes and 2 cart icon types.
  3. Compare – Allows you to design your compare process in simple and friendly way.
  4. Currency Swither – Needs for switching currency of whole site.
  5. Languate Swither – Needs for switching currency of whole site.
  6. Mobile Menu Toggle – Provide you perfect responsive options of Main-Menu.
  7. Vertical Divider – Make your header more logical and intuitive than ever before.
  8. Wishlist – UDesign Wishlist Element is very compatible with yith plugin.

UDesign Single Builder Elements

UDesgin Single Builder Elements is only supported for Single builder. You can build your own layout as much as possible using our following elements.

  1. Post Title – Display the title of current post.
  2. Post Date & Category – Display the date post added and post category.
  3. Post Excerpt – Show the short description of post.
  4. Post Content – Show off the content of the post.
  5. Featured Image – Show off the post feature image.
  6. Author Box – Display the author information.
  7. Post Comments – Show off the customer’s say to the site.
  8. Post navigation – Allows you to locate navigation on any part of the page.
  9. Related Posts – Display the related posts with current one.
  10. Post meta – Show off the post meta such as author name, comments count and so on.
  11. Post Tags – Show tags of current post.

UDesign Archive Builder Elements

UDesgin Archive Builder Elements is only supported for Archive builder. You can build your own layout as much as possible using our following elements.

  1. Post Title – Display the title of post archive page.
  2. Archive Posts – Display all posts added to current site.

UDesign Shop Builder Elements

UDesgin Shop Builder Elements is only supported for Shop builder. You can build your own shop layout as much as possible using our following elements.

  1. Shop Products – Display all products added to current site.
  2. Shop Sort – Place the sort options on any part of your page.
  3. Product Results – Show the number of products existed in current site.
  4. Pagination – Display the pagination on any part of your page.
  5. Count Per Page – Display options to set the number of products showed per page.
  6. Grid / List Toggle – Select a layout mode such as grid and list.
  7. Filter Toggle – Put the filter button in shop page.

UDesign Single Product Elements

UDesgin Single Product Elements is only supported for Single product layout builder. If you mind our single product page layout, then you can build your own layout using UDesign single product elements. All elements functionality is like this:

  1. Product Images – Show product gallery images with thumbnails.
  2. Product Navigation – Allows you to locate navigation on any part of page.
  3. Product Title – Display title of product.
  4. Product Meta – Display product meta including category and sku.
  5. Product Rating – Show off product ratings.
  6. Product Price – Display product price.
  7. Product Flash Sale – Show countdown in case of sale product.
  8. Product Excerpt – Show short description of product.
  9. Product Cart Form – Display quanityinput, add to cart and product variation.
  10. Product Share – Display share icons.
  11. Product Data tabs – Showcase product data such as description, specification or reviews.
  12. Product Frequently Bought Together – Display the products with which customers often buy current product.
  13. Linked Products – Show off related product.
  14. Product Compare – Allows you to insert compare button in any part of your single product page.
  15. Product Counter – Display sold count of product
  16. Product Wishlist – Allows you to add wishlist button to any part of your single product page.
  17. Product Tags – Show the tags added to current product.
