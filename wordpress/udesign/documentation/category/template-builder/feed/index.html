UDesign Template Builder is a set of builders that constitute a page. It helps you build every single part of the page faster and easily with drag & drop page builders. You can manage your templates at UDesign > Templates on WP Admin sidebar.

Why UDesign Template Builder?

Here are 3 reasons why UDesign Template Builder is useful:

  1. You can build any types of headers faster and more flexible by DND(drag and drop) with Elementor page builder.
  2. Comes with various pre-built templates.
  3. Dozen of builder elements with full-bodied options are ready for you.
  • Block Builder
  • Header Builder
  • Footer Builder
  • Popup Builder
  • Single Builder
  • Archive Builder
  • Single Product Builder
  • Shop Builder

UDesign Header Builder is one of template builders that UDesign provides. It helps you build any types of header faster and easily with Elementor page builder. You can manage your site headers at Header Builder section in UDesign > Templates page.

UDesign Header Builder Elements

Here is UDesign Header Builder elements list.

  • Cart Form: Display your cart status with mini-cart icon.
  • Language Switcher: Display available languages as dropdown menu.
  • Currency Switcher: Display available currencies as dropdown menu.
  • Mobile Menu Toggle: Display mobile menu toggle button on mobile.
  • Vertical Divider: Display colored vertical divider.
  • Account: Display user’s site login status and opens modal to login or register.
  • Wishlist: Display wishlist shortcut leading to page that shows wishlist products.
  • Compare: Display compare shortcut leading to page that shows products in compare list.

UDesign Popup Builder is one of template builders that UDesign provides. It helps you build any types of popup faster and easily with Elementor page builder. You can manage your site popups at Popup Builder section in UDesign > Templates page.

UDesign Popup Builder Settings

Here is UDesign Popup Builder settings list.

  • Width: Set popup’s width.
  • Height: Set popup’s height as it’s content or set it manually.
  • Content Position: Popup’s content alignment.
  • Position: Popup container’s position on screen.
  • Style Options: Includes overlay color, content background, shadow effect and etc.
  • Popup Animation: Select popup’s appear animation.
  • Animation Duration: Set popup’s animation duration.
